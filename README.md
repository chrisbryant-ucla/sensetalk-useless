# Useless SenseTalk Scripts

Scripts that are intentionally simple and "useless" in that they do not serve as functional snippets; rather they are fun and often humorous examples of how to program in SenseTalk.

## Scripts

### sensetalk-88-pints.script

A small SenseTalk demonstration inspired by the classic "99 Bottles of Beer" programming exercise. Instead of bottles of beer, this script logs verses about 88 pints of Guinness at the pub. There are 88 pints in a 50-liter keg. It demonstrates variables, a countdown repeat loop, conditional singular/plural wording, log output, and explicit loop termination.
